# SpiritVale Overlay

SpiritVale Overlay is a desktop companion for looking up official SpiritVale game-reference data while playing. It is read-only and remains separate from the game client.

## Language

**Command Palette**:
A compact, hotkey-toggleable overlay panel used to search the official game-data catalog and open a result's detail view.
_Avoid_: Sidebar, HUD, permanent overlay

## Example dialogue

Developer: “The Command Palette is open; search for a boss by name.”

Domain expert: “Select a result to open its official-data detail view, then toggle the Command Palette closed when finished.”
